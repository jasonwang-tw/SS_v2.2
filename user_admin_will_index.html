五種遺囑類別介紹

  • 公證遺囑

    經公證人公證的遺囑就可以算公證遺囑。
    一、要指定二人以上之見證人。
    二、要在公證人前口述遺囑旨意。
    三、要由公證人筆記、宣讀、講解經遺囑人認可。
    四、要記名年、月、日，由公證人、見證人及遺囑人同行簽名。如果遺囑人不能簽名的話，由公證人將其事由記名，就按指印代替簽名。如果公證人是法院所屬的公務員以及民間公證人，他所作成的文書即是公文書。在各地法院都設有公證處及民間公證人事務所，關於遺囑公證事宜只要找法院公證處及民間公證人士所所詢問即可。

    如果遺囑人行動不便，也可請公證人到指定之地方做公證遺囑，只需支付公證人之出差費用即可。(消提醒：要帶見證人一起去)

    公證人條件是：
    未成年人；禁治產人；繼承人及其配偶或其直系血親；受遺贈人及其配偶或其直系血親；為公證人或代行公證職務人之同居人、助理人或受僱人。

遺囑分配正確性

對於訂定遺囑應該要找誰協助訂定(或訂定為後檢查) ，有人找顧問，有人找代書，實際上找律師確認過遺囑的正確性與公平性是最為妥當。每個律師都可以幫你確認出遺囑的分配正確性，然而若遺囑的分配正確性要完全無誤，公證人才會幫你公證，所以就像契約一樣，每個文書若經由公證人公證後，此文書的公正性被執行的可能度會降至最低，日後被更動的幅度也越小。
